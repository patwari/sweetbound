Sweetbound

Sweetbound is a Phaser 3 game project scaffold that establishes the core scene flow and structure for a multi-biome platformer with land, water, and lava stages.

How to run locally:
1) Open index.html in a modern browser.
2) No build steps are required.

GitHub Pages deployment:
Push the repository to GitHub and enable GitHub Pages for the main branch; the game runs directly from index.html.
